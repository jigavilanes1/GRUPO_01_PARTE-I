JavaScript HTML DOM - Cambio de HTML

El HTML DOM permite que JavaScript cambie el contenido de los elementos HTML.

Cambio de contenido HTML

La forma más sencilla de modificar el contenido de un elemento HTML es mediante la propiedad innerHTML.Utlizando:

document.getElementById(id).innerHTML = new HTML


Ejemplo

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript puede Cambiar HTML </h2>
<p id="p1">Hola Mundo!</p>
<script>
document.getElementById("p1").innerHTML = "Texto Nuevo!";
</script>
</body>
</html>


JavaScript puede Cambiar HTML

Hola Mundo!


Ejemplo explicado:

  • El documento HTML anterior contiene un elemento <p> con id="p1"
  • Usamos el HTML DOM para obtener el elemento con id="p1"
  • Un JavaScript cambia el contenido ( innerHTML) de ese elemento a "¡Texto nuevo!"

Ejemplo 2

<!DOCTYPE html>
<html>
<body>
<h1 id="id01">Titulo Antiguo </h1>
<script>
const element= document.getElementById("id01");
element.innerHTML="Titulo Nuevo";
</script>
</body>
</html>


Titulo Antiguo

Cambiar el valor de un atributo

Para cambiar el valor de un atributo HTML, use esta sintaxis:

document.getElementById(id).attribute = new value


Ejemplo

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM </h2>
<img id="image" src="../img/loading.gif" width="160" height="120">
<script>
document.getElementById("image").src = "../img/landing.jpg";
</script>
</body>
</html>


JavaScript HTML DOM


Ejemplo explicado:

  • El documento HTML anterior contiene un <img> elemento con id="image"
  • Usamos el HTML DOM para obtener el elemento con id="image""
  • Un JavaScript cambia el src atributo de ese elemento de "s.gif" a "landscape.jpg"

Contenido HTML dinámico

JavaScript puede crear contenido HTML dinámico:


Ejemplo

<!DOCTYPE html>
<html>
<body>
<p id="demo"></p>
<script>
document.getElementById("demo").innerHTML = "Date : " + Date(); </script>
</body>
</html>


document.write()

En JavaScript, document.write() se puede usar para escribir directamente en el flujo de salida HTML:


Ejemplo

<!DOCTYPE html>
<html>
<body>
<p>Bla Bla Bla </p>
<script>
document.write(Date());
</script>
<p>Bla Bla Bla </p>
</body>
</html>
